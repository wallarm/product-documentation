# レポートの作成

イベントをフィルタリングしてから、結果をPDFまたはCSVレポートとしてエクスポートできます。Wallarmは作成したレポートを指定のメールアドレスに送ります。

PDFは視覚的に充実したレポートで、データ解析やプレゼンテーションに適しています。このレポートには次の内容が含まれます:

* 攻撃、脆弱性およびインシデントの概要
* イベントの詳細情報

CSVはフィルタに一致する各イベントの詳細を含み、技術的な目的に適しています。ダッシュボードの作成、ユニークな攻撃者IPの抽出、攻撃対象のAPIホスト／アプリケーションのリスト作成などに利用できます。

CSVレポートには複数のCSVファイルが含まれる場合があり、各イベントタイプ―攻撃、インシデント、脆弱性―ごとに1つのCSVファイルが生成されます。各CSVには最大10,000件のイベントが含まれ、ヒット数が多い順に並べられます。

## レポートの生成

Wallarm Consoleでは**Attacks**、**Incidents**または**Vulnerabilities**セクションからレポートを生成できます。使用するセクションに関わらず、レポートにはすべてのタイプのイベント―攻撃、インシデント、脆弱性―が含まれます。レポートの内容は現在適用されているフィルタに依存します。攻撃のフィルタはインシデントにも自動的に適用され、逆も同様です。脆弱性については、レポートに常に現在有効な脆弱性の一覧が含まれます。

レポートを生成するには:

1. Wallarm Consoleで**Attacks**、**Incidents**または**Vulnerabilities**セクションにアクセスします。
1. イベントを[フィルタ][link-using-search]します。
1. **Report**（または**Vulnerabilities**の場合は**PDF/CSV**）をクリックし、PDFまたはCSVを選択します。
1. **Send to**メールを設定します。

    ![レポート作成ウィンドウ][img-custom-report]
1. **Export**をクリックします。Wallarmはレポートを生成してメールで送ります。

## 以前のレポートのダウンロード

[脆弱性向けに生成された](../vulnerabilities.md#downloading-vulnerability-report)最新3件のPDFレポートは、生成日から6ヶ月間保存されます。

必要に応じて、エクスポートウィンドウからダウンロードしてください。

## メールで定期的なレポートの受信

PDFレポートを日次、週次または月次で定期的にメールで受信できます。このレポートには、該当期間の攻撃およびインシデントのデータと現在有効な脆弱性が含まれます。

レポートの受信の有無と頻度は、[メールレポート](../../user-guides/settings/integrations/email.md)統合機能の設定で構成してください。
