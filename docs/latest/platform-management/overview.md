# Wallarm Console

**Wallarm Console** is the unified web interface for managing Wallarm products. It is the place where you configure, monitor, and operate the products of [Wallarm AI Control Platform](../about-wallarm/overview.md) that are managed centrally:

* **[Wallarm API Security](../about-wallarm/api-security-overview.md)** — full operational surface: API discovery, attack monitoring, rules, integrations, mitigation controls, sessions, and more.
* **Wallarm Infrastructure Discovery** (AWS-only) — cross-account AWS asset discovery and the relationship graph (when active on the subscription).
* **[Wallarm API Security Testing](../vulnerability-detection/security-testing-overview.md)** — vulnerability findings, AASM results, and test runs.

**Wallarm AI Hypervisor** is managed on a separate domain and is not configured through Wallarm Console — see the [AI Hypervisor overview](../ai-hypervisor/overview.md) for details.

## What's included

* **[Plans & Pricing](../about-wallarm/subscription-plans.md)** — subscription plans and pricing options.

* **[Users & Access](../user-guides/settings/users.md)** — users, API tokens, SSO configuration, and LDAP integration.

* **[Monitoring & Events](../user-guides/events/overview.md)** — view and analyze attacks, incidents, and vulnerabilities across products.

* **[Dashboards](../user-guides/dashboards/threat-prevention.md)** — threat prevention, API Discovery, OWASP API Top 10, and BI dashboards.

* **[Triggers & Alerts](../user-guides/triggers/triggers.md)** — automated triggers and alert notifications.

* **[Search & Reports](../user-guides/search-and-filters/use-search.md)** — search events and generate custom reports.

* **[Account Settings](../user-guides/settings/account.md)** — your account, applications, subscriptions, and audit logs.
