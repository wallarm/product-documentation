# Wallarm API セキュリティ

Wallarm のソリューションは、OWASP API の上位10脅威、APIの不正利用、その他の自動化された脅威からAPI、マイクロサービス、Webアプリケーションを保護します。手動でルールを設定することなく、超低誤検出率でこれを実現します。
