* البنية التحتية بتاعتك مبنية على أجهزة فعلية أو آلات افتراضية من غير استخدام طرق بناء على الحاويات. عادة، الإعدادات دي بيتم إدارتها بأدوات Infrastructure as Code (IaC) زي Ansible أو SaltStack.
