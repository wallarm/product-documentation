Restart NGINX using the following command:

=== "Debian"
    ```bash
    sudo systemctl restart nginx
    ```
=== "Ubuntu"
    ```bash
    sudo systemctl restart nginx
    ```
=== "CentOS"
    ```bash
    sudo systemctl restart nginx
    ```
=== "AlmaLinux, Rocky Linux or Oracle Linux"
    ```bash
    sudo systemctl restart nginx
    ```
=== "RHEL"
    ```bash
    sudo systemctl restart nginx
    ```
=== "Alpine Linux"
    ```bash
    sudo rc-service nginx restart
    ```
