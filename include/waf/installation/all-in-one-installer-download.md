Wallarm suggests all-in-one installations for the following processors:

* x86_64
* ARM64

To download all-in-one Wallarm installation script, execute the command:

=== "x86_64 version"
    ```bash
    curl -O https://meganode.wallarm.com/6.8/wallarm-6.9.0.x86_64-glibc.sh
    ```
=== "ARM64 version"
    ```bash
    curl -O https://meganode.wallarm.com/6.8/wallarm-6.9.0.aarch64-glibc.sh
    ```
