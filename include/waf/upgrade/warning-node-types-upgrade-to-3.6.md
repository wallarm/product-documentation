!!! warning "Breaking changes and recommendations for upgrading different node types"
    * If upgrading Wallarm node 2.18 or lower, note that version 3.x contains **breaking changes**. Before upgrading the modules of 2.18 and lower up to 3.6, carefully review the list of [Wallarm node changes](older-versions/what-is-new.md) and consider a possible configuration change.
    * We recommend upgrading both the regular (client) and [multi-tenant](../installation/multi-tenant/overview.md) nodes up to version 3.6. It allows you to stay up to date with Wallarm releases and prevent [installed module deprecation](versioning-policy.md#version-support).
