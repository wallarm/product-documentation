# Doc Agent Reference Files

This directory contains reference materials for the documentation agent.
These files are read by Claude when generating documentation.

## How to update

### Style guide
Export from Confluence as Markdown and save as `style-guide.md`:
1. Open the style guide page in Confluence
2. Export → Markdown (or copy-paste the content)
3. Save here as `style-guide.md`

### Glossary
Export from Confluence as Markdown and save as `glossary.md`:
1. Open the glossary page in Confluence
2. Export → Markdown (or copy-paste the content)
3. Save here as `glossary.md`

The agent will reference these files when generating documentation to ensure
consistent terminology and writing style.
